# 解决在IDEA中控制台中文乱码

1. IDEA安装目录/bin/idea.exe.vmoptions 或者（64位为idea64.exe.vmoptions）在文件末尾加上 -Dfile.encoding=UTF-8
2. Run/Debug Configuration--->server--->VM options 设置为file.encoding=UTF-8
